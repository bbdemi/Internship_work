# Embedded AI Workflows

This repository documents my learning about AI-assisted coding workflows for embedded systems, robotics, ESP32, Arduino, and C++.

## Goal

The goal is to learn how to use AI tools like ChatGPT, Claude Code, and Codex to write cleaner, safer, and more production-level embedded code.

I want to build better coding habits for embedded and robotics projects by using AI properly instead of only asking random questions.
