# Urban Monkey Stepper Motor Accuracy Experiment

## Objective

The objective of this experiment was to test how accurately the Urban Monkey robot rotates when a rotation command is given to the stepper motor.

The experiment was done to compare the expected rotation angle with the actual rotation angle and understand the error in the robot’s rotational movement.

---

## Work Done

- Conducted a stepper motor rotation accuracy experiment on the Urban Monkey robot.
- Gave specific rotation commands to the robot.
- Observed the actual physical rotation of the robot.
- Compared the commanded rotation with the measured rotation.
- Noted the difference/error between expected and actual rotation.
- Documented the observations for further analysis.

---

## Components / Tools Used

- Urban Monkey robot setup
- Stepper motor
- Stepper motor driver
- Power supply
- ESP32 / controller board
- Measuring scale / angle marking setup
- Multimeter
- Jumper wires and connectors
- Laptop with Linux
- Arduino IDE / terminal tools

---

## Experiment Method

1. The Urban Monkey robot was placed in a fixed position.
2. A rotation command was given to the stepper motor.
3. The expected rotation angle was noted.
4. The actual rotation of the robot was measured physically.
5. The error was calculated by comparing expected and actual rotation.
6. Multiple readings were taken to observe consistency and accuracy.

---

## Error Calculation

Error = Actual Angle - Commanded Angle
